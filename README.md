# Dielectric-Properties-plotting
Browser-based scientific plotter for HIOKI IM35xx LCR meter data. Origin-style EIS plots (Nyquist, Bode, Cp, Rs, D) with 3D gradient markers, up to 4 colored Y axes, cross-sample comparison, WYSIWYG PNG/SVG export, CSV-zip for Origin import, and self-contained HTML session save.
